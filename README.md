# AI-Driven Anomaly Detection for Cloud Authentication Security

## 🔍 Overview
This project presents a machine learning-based anomaly detection system designed to identify suspicious login behaviour in cloud authentication environments. The system analyses behavioural patterns such as login time, device usage, and IP variability to detect potential security threats.

## 🧠 Key Features
- Behaviour-based anomaly detection using machine learning
- Feature engineering for user login patterns
- Class imbalance handling using `class_weight='balanced'`
- Hyperparameter tuning using GridSearchCV
- Evaluation using precision, recall, and F1-score
- Designed for real-time integration via Flask API (conceptual)

## ⚙️ Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib
- Google Colab

## 📊 Dataset
A synthetic cloud authentication dataset was used to simulate real-world login behaviour and anomaly scenarios.

## 📈 Results
The model demonstrates strong performance in detecting anomalous login attempts, achieving balanced precision and recall through proper feature engineering and model tuning.
