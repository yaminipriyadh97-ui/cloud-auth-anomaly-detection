# AI-Driven Anomaly Detection for Cloud Authentication Security

## Overview
This project presents a machine learning-based anomaly detection system for identifying suspicious login behaviour in cloud authentication environments.

## Features
- Behaviour-based anomaly detection
- Machine learning model (Random Forest)
- Feature engineering (login patterns, device consistency, IP variability)
- Model evaluation (precision, recall, F1-score)
- Flask API for real-time prediction (conceptual design)

## Tools & Technologies
- Python
- Scikit-learn
- Pandas, NumPy
- Flask
- Google Colab

## Dataset
Synthetic authentication dataset used for experimentation and evaluation.

## Future Work
- Extend to agentic AI system for autonomous response
- Integrate multi-agent architecture
- Real-time deployment in cloud environments
